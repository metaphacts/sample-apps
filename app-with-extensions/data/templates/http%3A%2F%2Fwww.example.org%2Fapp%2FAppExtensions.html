1. App with extensions

App with extensions

This app show-cases some extension mechanisms provided by the platform:

  • Additional helper to show the current date and time: [[exampleDateTime]]
  • A REST endpoint /rest/extension/custom/connection-check which performs a simple SPARQL query on the default repository
  • A HTML5 component saying hello:
  • A dynamic permission provider for assets which reads permissions from a property file

Dynamic permission provider for assets

The property file to configure the provider must reside in config/dynamicPermissions.prop of the configured security storage (see securityConfigStorageId). Note that the securityConfigStorageId is not configured by default.

Here is an example of such a property file:

myOntology.iri=https://ontologies.metaphacts.com/owner-test/0.1 myOntology.userName=tester myOntology.roles=owner,author myOntology.type=Ontology myVocabulary.iri=https://vocabularies.metaphacts.com/owner-test/0.1 myVocabulary.userName=tester myVocabulary.roles=owner myVocabulary.type=Vocabulary myOntology2.iri=https://ontologies.metaphacts.com/author-test/0.1 myOntology2.userName=tester myOntology2.roles=author myOntology2.type=Ontology myVocabulary2.iri=https://vocabularies.metaphacts.com/author-test/0.1 myVocabulary2.userName=tester myVocabulary2.roles=author myVocabulary2.type=Vocabulary
